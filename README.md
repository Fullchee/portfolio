# portfolio

Made with

- Next.js
- Tailwind
- TypeScript

Inspired by

- Josh Comeau's [Building an Effective Dev Portfolio](https://www.joshwcomeau.com/effective-portfolio/)

1. `npm i`
2. `npm run dev`

`git push origin main`
