# portfolio

This portfolio was made with

- Next.js
- Tailwind CSS 2
- TypeScript

The design is inspired by

- Josh Comeau's [Building an Effective Dev Portfolio](https://www.joshwcomeau.com/effective-portfolio/)
- juliacodes.com

## Install

1. Install yarn v1
   1. `brew install yarn`
2. `yarn build`
3. `yarn dev`

## Deploy

`git push origin main`
