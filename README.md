# portfolio

Made with

- Next.js
- Tailwind CSS 2
- TypeScript

Inspired by

- Josh Comeau's [Building an Effective Dev Portfolio](https://www.joshwcomeau.com/effective-portfolio/)
- juliacodes.com

## Install

1. Install yarn v1
   1. `brew install yarn`
2. `yarn build`
3. `yarn dev`
4. Setup Sentry
   1. `sentry-cli login`

## Deploy

`git push origin main`
